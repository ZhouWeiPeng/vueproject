# pure-music
#### 项目预览图：
![预览图失效](https://static01.imgkr.com/temp/e3ea667c2dda4b36928a385b7833c987.gif)
#### 先运行node服务器（api是node.js版的）
#### api地址：(https://github.com/Binaryify/NeteaseCloudMusicApi)
#### 进入 /NeteaseCloudMusicApi 安装node服务器所需插件：
```
npm i
```

#### 运行node服务器：
```
node app.js
```

#### 返回根目录安装项目所需插件：
```
npm i
```

#### 最后运行
```
yarn serve
```
